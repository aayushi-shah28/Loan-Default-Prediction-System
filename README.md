# Loan Default Prediction System

## Project Overview

This project focuses on predicting loan approval outcomes using machine learning techniques. A dataset of 614 historical loan applications was analyzed to identify key factors influencing loan decisions and build predictive models using Logistic Regression and Random Forest for risk assessment.

## Dataset

* 614 historical loan application records

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib

## Methodology

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Logistic Regression
* Random Forest Classification
* Model Evaluation using Accuracy, Precision, Recall, and F1-Score

## Results

* Achieved 78.9% accuracy in predicting loan approval outcomes
* Evaluated Logistic Regression and Random Forest models
* Identified Credit History, Applicant Income, and Loan Amount as key factors influencing loan approval decisions

## Project Outcome

The project demonstrates the application of machine learning techniques to support data-driven lending decisions and credit risk assessment.
